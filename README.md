# Simple CLI to use ChatGPT with Node.js

## CLI example

### Requirements

- node in version >= 18

### Pre

```sh
echo "OPENAI_API_KEY=\"sk-TODO\"" > .env # your key of chatgpt in https://platform.openai.com/account/api-keys
npm install
chmod +x index.js
```

### Using

```sh
./index.js who born first, the chicken or the egg\?
```
